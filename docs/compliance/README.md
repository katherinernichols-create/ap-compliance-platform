# AP Worker Compliance Requirements

## Quick Reference: Mandatory Credentials

### Every Worker Must Have

**Screening (ONE of):**
- NDIS Clearance (5 years validity) OR
- Police Check (3 years validity)

**Training (Annual):**
- Current CPR Certification (12 months)
- Manual Handling (12 months)  
- Infection Control (12 months)
- Code of Conduct training
- SIRS training

**Role-Specific:**
- PCAs: Certificate III in Individual Support
- RNs: AHPRA Registration (annual renewal)
